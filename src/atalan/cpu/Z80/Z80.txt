****
Z80
****

Support for Z80 is currently far from being complete. It can print texts and numbers,
do some basic arithmetics, loops and conditions.


=========
Assembler
=========

The Atalan generated asm source code is compiled by 
<a href="http://www.arrakis.es/~ninsesabe/pasmo/">Pasmo Assembler</a> compiler. 

=========
Reference
=========

For more information about Z80 processor go to <a href="http://www.z80.info/">http://www.z80.info/</a>.
